# AWOS Recruitment — Product Summary

## Vision

To provide AI coding assistants with a zero-setup, intelligent discovery engine that dynamically matches the right skills, agents, and tools to the developer's specific needs — eliminating local dependency management, ensuring consistent AI tooling across distributed teams, and providing visibility into which AI capabilities drive real-world value.

## Target Audience

Software developers and engineering teams using AI coding assistants (e.g., Claude Code) who need reliable, discoverable tooling without manual setup.

## Core Features

- **Semantic Capability Search** — Natural language queries matched via embeddings to a centralized registry.
- **Capability Registry & Indexing** — Git-managed catalog of skills, agents, and tools with rich metadata.
- **Client-Initiated Discovery** — All discovery driven by client search queries; server is stateless regarding project context.
- **Capability Installation** — Install discovered capabilities directly via an npx package.
- **Usage Telemetry & Analytics** — Track discovery, installation, and usage for data-driven tooling decisions.
- **MCP Protocol Integration** — Native MCP server interface for zero-setup client connectivity.
